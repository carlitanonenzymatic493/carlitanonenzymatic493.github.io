## 📌 What is claude-code-recap?

This tool helps you keep track of your work when you use Claude Code. Many people code across different projects. It becomes hard to remember where you left off. This app solves that problem. It lists your past sessions in one place. You see the project location, a short summary of the work, the git branch used, the artificial intelligence model, and how many turns the conversation took.

The tool provides a command you copy and paste to resume your work exactly where you stopped. You also have the option to open an entire working set in your terminal tabs with one command. Your data stays on your computer. This tool does not send information over the internet. No tracking happens.

## 🛠️ How to Use the Application

Once you install the software, you can open it like any other program. 

### Viewing Sessions
When you open the app, it scans your computer for session history logs created by Claude Code. The main window displays a list of these sessions. You can scroll through the list to find the project you want.

### Understanding the Details
Each entry in the list shows:
- Project Path: The folder where your code lives.
- Summary: A brief note on what you did.
- Git Branch: The specific version of your code.
- Model: Which version of the AI you used.
- Turn Count: How many back-and-forth messages occurred.

### Resuming Work
Each session has a button labeled "Copy Resume Command." Click this button to copy the text to your clipboard. Open your terminal or command prompt, paste the text, and press Enter. This launches Claude Code and restores your session state.

### Opening Multiple Tabs
If you worked on several files at once, use the "Open Set" button. The app automatically triggers your computer to open the necessary terminal tabs for that workspace. This saves time and ensures you have all your tools ready.

## 🔒 Privacy and Security

You might worry about your code history. This tool operates entirely offline. It reads local files created during your coding sessions. It does not contain code to connect to any servers. It does not send your project summaries or personal data to any company. We believe that your developer history belongs to you. Since no network requests happen, your environment stays secure.

## ❓ Frequently Asked Questions

**Does this app work with Mac or Linux?**
This version is designed specifically for Windows. 

**Does the app modify my code?**
No. The app only reads your session logs. It makes no changes to your code files or your git repositories.

**What happens if I delete a log file?**
The app will simply stop showing that session in the list. It will not break your project or delete your code.

**Where does the app store my data?**
It reads from the directories where Claude Code saves its history by default. You can see these locations in the settings menu of the app.
